# Portfolio Website

A modern, responsive portfolio website for Shafiqullah Khan - Frontend Web Developer & Software Engineering Student.

## 🌟 Features

- **Modern Design**: Dark theme with glassmorphism effects and vibrant gradients
- **Responsive**: Fully responsive design that works on all devices
- **Interactive**: Smooth animations, typing effects, and scroll-triggered animations
- **SEO Optimized**: Proper meta tags and semantic HTML
- **Fast**: Optimized performance with vanilla CSS and JavaScript

## 🚀 Technologies Used

- HTML5
- CSS3 (Glassmorphism, Gradients, Animations)
- Vanilla JavaScript
- Font Awesome Icons
- Google Fonts (Inter)

## 📂 Project Structure

```
portfolio/
├── index.html          # Main HTML file
├── style.css           # Stylesheet with modern design
├── script.js           # Interactive functionality
└── README.md           # Project documentation
```

## 🎨 Sections

1. **Hero Section** - Eye-catching introduction with typing animation
2. **About** - Professional summary and contact info
3. **Skills** - Categorized technical skills
4. **Projects** - Featured project showcase
5. **Experience** - Work history timeline
6. **Education** - Academic background
7. **Contact** - Contact information and social links

## 🛠️ How to Run

Simply open `index.html` in your web browser. No build process required!
